# Calculator

CutefishOS Calculator

## Dependencies

For Debian:
```shell
sudo apt-get install cmake extra-cmake-modules libkf5windowsystem-dev libqt5x11extras5-dev qtbase5-dev qtdeclarative5-dev qtquickcontrols2-5-dev qttools5-dev qttools5-dev-tools
```

## Build and Install

```
mkdir build
cd build
cmake ..
make
sudo make install
```
